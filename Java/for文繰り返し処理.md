# 同じ処理を何度も繰り返す  

public class Main {  
　　public static void main(String[] args ) {  
　　　　String greeting = "Hello world";  

　　　　for (int i = 0; i < 5; i++) {  
　　　　　　System.out.println("greeting");  
　　　　}  
　　}  
}  
## for (int i = 0; i < 5l i++)  
### 初期値　i = 0;  
### 繰り返し条件　i < 5;  
### 変化文　i++  

###　カウンター変数を表示する  
System.out.println(greeting + ":" + i);
